# yabred/homebrew

Personal Homebrew tap for macOS apps by [yabred](https://github.com/yabred).

## Setup

```bash
brew tap yabred/homebrew
```

## Apps

### FROK

Instant sound effects for macOS from the menu bar, hotkeys, or the command line.

```bash
brew install --cask frok
```

See the [FROK repository](https://github.com/yabred/FROK) for usage and troubleshooting.
